## O que e Git?
Git e um sistema de controle de versao distribuido que permite rastrear alteracoes em arquivos ao longo do tempo.
Com ele, e possivel trabalhar em equipe sem sobrescrever o trabalho dos outros.

## O que e GitHub?
GitHub e uma plataforma de hospedagem de repositorios Git que facilita a colaboracao entre desenvolvedores.
Oferece recursos como Issues, Pull Requests, Actions e muito mais.

## Fluxo de Trabalho Basico
1. git clone — Clonar o repositorio
2. git checkout -b nome-da-branch — Criar uma nova branch
3. git add . — Adicionar alteracoes
4. git commit -m "mensagem" — Salvar alteracoes localmente
5. git push — Enviar para o GitHub
6. Abrir um **Pull Request** para revisao
