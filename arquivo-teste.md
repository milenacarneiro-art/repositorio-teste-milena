## O que e um Branch?

Um branch (ramificacao) e uma linha independente de desenvolvimento no Git.
Ele permite trabalhar em novas funcionalidades ou correcoes sem afetar o codigo principal (main).

### Comandos de Branch
`ash
# Criar uma nova branch
git checkout -b minha-branch

# Listar todas as branches
git branch

# Trocar de branch
git checkout nome-da-branch

# Deletar uma branch
git branch -d nome-da-branch
`

## O que e um Commit?

Um commit e um registro permanente das alteracoes feitas nos arquivos.
Cada commit possui um identificador unico (hash) e uma mensagem descritiva.

### Boas praticas para mensagens de commit
- Use verbos no imperativo: "adiciona", "corrige", "remove"
- Seja direto e descritivo
- Exemplos:
  - eat: adiciona tela de login
  - ix: corrige erro de validacao no formulario
  - docs: atualiza README com instrucoes de instalacao
