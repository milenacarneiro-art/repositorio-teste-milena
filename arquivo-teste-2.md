## O que e um Pull Request?

Um Pull Request (PR) e uma solicitacao para integrar as alteracoes de uma branch em outra.
E o principal mecanismo de revisao de codigo em equipe no GitHub.

### Como abrir um Pull Request
1. Faca push da sua branch para o GitHub
2. Acesse o repositorio no GitHub
3. Clique em **"Compare & pull request"**
4. Adicione um titulo e descricao clara
5. Solicite revisores se necessario
6. Clique em **"Create pull request"**

### O que e um Fork?

Fork e uma copia de um repositorio na sua propria conta do GitHub.
Permite contribuir com projetos de outras pessoas sem ter acesso direto ao repositorio original.

Fluxo de contribuicao com Fork:
`
Repositorio Original → Fork (sua conta) → Branch → Commits → Pull Request → Merge
`
